# MohsinApex-MD – WhatsApp Bot

**MohsinApex-MD** is a powerful WhatsApp bot built by **Mohsin Raza**.  
It supports multi‑device, AI, downloads, group management, games, economy, and a web dashboard.

## 🚀 Deploy on Railway

1. Push this code to GitHub.
2. On Railway, create a new project from your repo.
3. Set environment variables (see .env.example).
4. Deploy – the bot will start and show QR/pairing code.
